Managing Structure Map File Descriptions


You manage the structure map file naming scheme descriptions through the following menu option:


The following screen will be displayed:


File prefix/suffix What the system will be looking for in the file name.
Position Where the prefix/suffix will be found (beginning/end)
Mandatory? Is it compulsory to have this structure map item.
Note that this only applies when the system is adding structures automatically.  If there are mandatory items and the system does not find any file matching the file specification then there will be no automatic structure map added.
You may manually add a structure without items defined as compulsory here.
Allow Multiples? Are we allowed to have more than one of these structure map items?
Note that this only applies when the system is adding structures automatically.  If the system finds multiple matches that aren't allowed then there will be no automatic structure map added.
You may manually add multiple structure items even if they are not allowed here.
