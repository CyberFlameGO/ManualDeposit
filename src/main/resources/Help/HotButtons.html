Hot Buttons

Select Files pane

When Intellectual Entity is not set

<Alt>M Display file menu (same as right click)
<Alt>S Selects current directory as the root of the Intellectual Entity
(Option 1 of menu)
<Alt>E Adds each file as an Intellectual Entity
(Option 2 of menu - if available)

When Intellectual Entity is set

Select a file or group of files, Press <ALT> and the first letter of a representation type.  This will add a new representation type to the Intellectual Entity with the selected files added.
Current representation types are "Digital Original", "Preservation Copy", "Modified Master", "Access Copy"

Intellectual Entity pane

<Delete> Deletes the selected file or directory from the IE
<Alt><Up> Moves the selected file or directory up
<Alt><Down> Moves the selected file or directory down
<Alt>S Takes all selected files and creates a new structure map item with those files.
<Alt>R Takes the first selected file, determines all other files that are different representations of the file, selects those files, then creates a new structure map item with them.

Structure Map pane

<Delete> Deletes the selected file or structure item from the structure map
<Alt><Up> Moves the selected file or structure item up
<Alt><Down> Moves the selected file or structure item down
